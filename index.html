🛠️STEP 1: 事前準備 - 必要なソフトを揃えよう

結論：3つの無料ソフトをMacにダウンロードします

バーチャル配信を始めるために、以下の3つのソフトウェアを公式サイトからダウンロードし、インストールしておきましょう。

  • VRoid Studio: オリジナルアバターを作る粘土細工のようなアプリ
  • 3tene: 作ったアバターに命を吹き込み、動かすためのアプリ
  • VCam: 3teneの映像をZoomに送るための「魔法のカメラ」アプリ

🍎 Mac環境の特殊事情

Mac、特に新しいM1/M2/M3チップ搭載モデルでは、Windowsと比べてセキュリティ設定が厳しくなっています。そのため、アプリの初回起動時にいくつかの特別な手順が必要になる場合があります。このガイドで詳しく説明するのでご安心ください。

具体的なダウンロード手順

  1. VRoid Studioをダウンロード
    「VRoid Studio」で検索し、公式サイト(vroid.com)からMac版をダウンロード。dmgファイルを開き、アプリを「Applications」フォルダにドラッグします。
  2. 3teneをダウンロード
    「3tene」で検索し、公式サイト(3tene.com)からMac版をダウンロード。同様にインストールします。
  3. VCamをダウンロード
    「VCam Mac」で検索し、公式サイト(GitHubページなど)からダウンロード。これも同様にインストールします。

この準備は、料理教室で「包丁」「まな板」「フライパン」を用意するようなものです。道具が揃えば、あとはレシピ通りに進めるだけです！

🧑‍🎨STEP 2: アバター作成 - VRoid Studioで自分だけのキャラクターを作ろう

結論：顔・髪・服を順番に選んで、VRMファイルで保存します

アバター作成の4ステップ

flowchart TD A[🚀 VRoid起動] --> B[👤 顔・髪型を編集] B --> C[👕 衣装をコーディネート] C --> D[💾 VRM形式でエクスポート] D --> E[✅ オリジナルアバター完成!]
  1. 顔・髪型をカスタマイズ
    VRoid Studioを起動し、「新規作成」からキャラクターのベースを選択します。画面左のメニューから「顔」「髪型」を選び、右側のスライダーやプリセットを使って、目、鼻、口の形や髪型、髪色を自由に調整します。
  2. 衣装をコーディネート
    左メニューの「衣装」タブで、トップス、ボトムス、靴などを選びます。各パーツの色や模様も「テクスチャ編集」で変更可能です。
  3. VRM形式でエクスポート（保存）
    アバターが完成したら、画面右上のエクスポートボタンをクリックします。「VRMエクスポート」を選択し、ファイル名を入力してデスクトップなど分かりやすい場所に保存します。この .vrm ファイルがあなたのアバターのデータになります。

🖱️ Macでの操作のコツ

  • ズーム: トラックパッドを2本指で上下にスワイプ
  • 回転: トラックパッドを2本指で押しながらドラッグ
  • 移動: Shiftキーを押しながら2本指でドラッグ

アバター作りは「デジタル粘土細工」や「着せ替え人形」のようなものです。正解はないので、楽しみながら理想のキャラクターを創造してみてください。

🏃‍♀️STEP 3: アバターを動かす - 3teneに命を吹き込もう

結論：作成したVRMファイルを3teneに読み込み、カメラで顔の動きを連動させます

⚠️ Macでの3tene起動時の最重要ポイント

Macのセキュリティ機能により、ダウンロードした3teneはそのままでは起動できないことがあります。以下の手順で起動してください。

Finderで「3teneFREE」アプリを探し、Controlキーを押しながらクリックし、表示されたメニューから「開く」を選択します。警告が出ても「開く」をクリックしてください。

アバターを動かす3ステップ

  1. 作成したアバター(VRM)を読み込む
    3teneを起動し、PC上のファイルをロードボタンをクリック。先ほどVRoid Studioで作成した .vrm ファイルを選択します。
  2. フェイストラッキングをONにする
    右側のメニューからフェイストラッキングをONにします。カメラへのアクセス許可を求められたら「許可」してください。MacのWebカメラが起動し、あなたの顔の動きがアバターに反映されます。
  3. 動きを確認する
    顔を上下左右に動かしたり、まばたきや口の開閉をしてみてください。アバターが同じように動けば成功です！

🤔 3teneが起動しない・動かない場合

M2 Macなど特定の環境では起動に失敗することがあります。その際は以下の「緊急起動モード」を試してください。

  • Altキー起動: Alt(Option)キーを押しながら3teneのアイコンをダブルクリックする。
  • 完全再インストール: アプリを完全に削除し、Macを再起動してから再ダウンロードする。

3teneの役割は「人形遣い」です。Webカメラという見えない糸で、あなた（人形遣い）の表情や動きをアバター（人形）に伝えています。

💻STEP 4: Zoomに映す - VCamを使ってバーチャル会議に参加しよう

結論：VCamを「仮想カメラ」として起動し、Zoomのカメラ設定でVCamを選択します

🤔 なぜVCamが必要なの？

Zoomは直接3teneの画面を映すことができません。そこでVCamという仲介アプリを使います。VCamは3teneの映像を受け取り、それを「本物のWebカメラからの映像」のように見せかけてZoomに送ってくれます。この仕組みを仮想カメラと呼びます。

Zoom連携の仕組み

graph LR subgraph Macの中 A(あなた 🧍) -->|顔認識| B(3tene 🏃‍♀️); B -->|映像を送る| C(VCam 📹); end C -->|仮想カメラとして認識される| D(Zoom 💻);

Zoomにアバターを映す4ステップ

  1. VCamを起動し、アバターを読み込む
    まずVCamを起動します。メニューの「ファイル」から「VRMファイルを開く」を選び、先ほどと同じ .vrm ファイルを読み込みます。VCamの画面にアバターが表示され、顔の動きに連動することを確認します。
  2. 【重要】Camera Extensionを有効化 (macOS 15以降)

    新しいmacOSでは、手動でVCamをカメラとして許可する必要があります。

    1. システム設定 → 一般 → ログイン項目と拡張機能 を開く。
    2. 「許可された機能拡張」セクションにある カメラ機能拡張 をクリック。
    3. VCam.app を見つけ、トグルをONにする。

    これをしないと、Zoomのカメラ一覧にVCamが表示されません。

  3. Zoomを起動し、設定を開く
    VCamと3teneを起動したまま、Zoomアプリを起動します。Zoomの設定（歯車アイコン）を開き、ビデオタブに移動します。
  4. カメラを「VCam」に切り替える
    「カメラ」という項目にあるドロップダウンメニューをクリックすると、カメラの一覧が表示されます。その中から「VCam」を選択します。プレビュー画面にあなたのアバターが表示されたら成功です！そのまま会議に参加すれば、アバターの姿で会話できます。

🤔 Zoomのカメラ一覧に「VCam」が出ない！

最も多いトラブルです。以下の順番で確認してください。

  1. VCamアプリが先に起動しているか確認。
  2. macOS 15以降の場合、Camera Extensionが有効になっているか再確認。
  3. Zoomアプリを完全に終了（Command+Q）してから、もう一度起動してみる。
  4. 最終手段として、Mac自体を再起動する。
